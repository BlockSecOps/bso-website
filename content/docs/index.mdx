## What is BlockSecOps?

BlockSecOps is a unified DevSecOps platform for Web3 that integrates 11+ security scanners including SolidityDefend, Slither, Aderyn, Semgrep, Solhint, Halmos, Echidna, Wake, and Medusa with 580+ detectors into a single, comprehensive security solution for smart contract development teams.

## Key Features

- **Unified Dashboard** - Centralized view of all security findings
- **11+ Integrated Scanners** - SolidityDefend, Slither, Aderyn, Semgrep, Solhint, Halmos, Echidna, Wake, Medusa, and more
- **580+ Detectors** - Comprehensive vulnerability detection
- **Deduplicated Findings** - AI-powered deduplication and correlation
- **CI/CD Integration** - Seamless integration with your development workflow
- **Multi-Chain Support** - Ethereum, Solana, and more
